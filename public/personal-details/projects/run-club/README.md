# Out of Office Run Club

Two-page website for a Sunday morning social running community based in Kharadi, Pune.

## Stack

Next.js 14 (App Router), TypeScript, Tailwind CSS, Netlify

## Pages

- **Home** — Hero, next-run bib card, club intro, photo grid, WhatsApp CTA
- **Schedule** — Run calendar with completed/upcoming runs, current route info

## Impact

- 40+ runners at launch
- Bi-weekly 5K social runs
- Deployed on Netlify with Next.js runtime

## Live Site

https://out-of-office-run-club.netlify.app
